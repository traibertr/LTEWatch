queue< _Tp, MAX_SIZE >::queue ( const _Tp &  nullObject)
inline

Create a queue base on a ringbuffer of MAX_SIZE elements.

The oldest element is lost if the queue is full. The queue memory is allocated on the stack and filled with null (empty) objects.

Parameters
nullObjectthe null/empty object to indicates the queue is free

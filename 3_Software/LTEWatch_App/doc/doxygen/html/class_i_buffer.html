size_t *  size 
)
inline

Append new incoming data.

Append new incoming data to the ones already in the buffer. If the pointer to the data to be added is null, the methods will return BUFFER_ERROR_NULL erro code. If the buffer is full or only a few data could be added to the current ones, the method returns BUFFER_ERROR_NO_MEM, otherwise it returns BUFFER_SUCCESS. In case of BUFFER_ERROR_NO_MEM result, the amount of data that has been added is provided by size parameter.

Parameters
dataPointer to the new values to append
sizeAmount of data to append to the buffer
Return values
BUFFER_SUCCESSFor a successfull process
BUFFER_ERROR_NO_MEMIf there is not enough room keft in the buffer
BUFFER_ERROR_NULLIf the buffer is full

According to the given parameters, this method updates to offset of the buffer.

It updates the current offset position of the buffer with the quantity of handled bytes len. According to the MTU max size of the current connection, the returns the size of the next chunk to deal with.

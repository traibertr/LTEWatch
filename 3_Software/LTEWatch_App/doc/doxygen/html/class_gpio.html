Abstraction of a gpio.

Gpio

This class is the Abstraction of a watch's gpio. Those gpio are defined by an identifier (ID) and a state to let the appication know on which a special pressure or release has been applied.

The behaviour of the gpio or the user's actions applied on it are controlled by a GpioController. See its documentation for more details.

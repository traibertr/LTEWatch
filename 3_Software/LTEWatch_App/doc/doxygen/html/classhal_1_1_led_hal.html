Detailed Description

Class of Hardware Abstraction Layer to handle LEDs of device based on the nRf5 series chips.

This file contains all necessary functions to handle LEDS on boards based on the nRF5 series and SDK v14.2.0 and above. It starts with the function initLEDs() to initialize the GPIOs used for the LEDs.

Detailed Description

This class is the controller of gpios used as inputs.

GPIO Controller

De-bouncer

to be sure that a pressure has been applied on a gpio, we have implemented a de-bouncer which use an app_timer with the parameters:

  • APP_GPIO_HANDLER_MS_TIMER_TICK: The Tick duration used by the timer of the gpio's debouncer.
  • GPIO_DEBOUNCE_MS_DELAY: The debounce time [ms].

In other words when an activity is detected on a gpio, it must be applied at least for a certain time at least to be considered as a real gpio's action. As soon as the debounce time is over the controller keep and increase the duration of each gpio's action until a release and determines if it's a single click, a long or very long press that has been applied.

Notification

When the gesture is identified, the controller will notify the observer of that fact !

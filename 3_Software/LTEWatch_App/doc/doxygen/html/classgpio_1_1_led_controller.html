◆ initialize()

void LedController::initialize ( uint8_t  index,
uint32_t  ticksOn = k_ms_to_ticks_near32(125),
uint32_t  totalTimeTicks = k_ms_to_ticks_near32(1000) 
)

Initializes the components needed by this class.

This initialization method needs at least one parameter, which is the identifier of the LED to be controlled. The different timing ON and OFF time can also be given here. If they are not defined, the default value of 1 second is applied for both of them.

Parameters
indexThe identifier of the LED to control
ticksOnThe delay, in terms of timer's ticks, the led should be ON, default value is 125 [ms]
totalTimeTicksThe duration of a single sequence, default value is 1000 [ms]
Warning
Must be called before any other method of this class
Note
Requires pre-existing instances of classes listed in parameter list

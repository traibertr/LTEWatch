void uGnssPosDateGetStop ( uDeviceHandle_t  gnssHandle)

Cancel a uGnssPosGetStart(); after this function has returned the callback passed to uGnssPosGetStart() will not be called until another uGnssPosGetStart() is begun. uGnssPosGetStart() also creates a mutex for thread safety which will remain in the system even after pCallback has been called; this will free the memory it occupies.

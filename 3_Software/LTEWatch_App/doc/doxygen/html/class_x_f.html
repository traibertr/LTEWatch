Detailed Description

Execution Framework.

Introduction

This library implements an eXecution Framework (XF) to drive finite state machines. Using this framework, it is possible to drive multiple state machines in a concurrent manner (pseudo-parallel) without having an operation system.

The XF defines an interface which gives the ability to reimplement parts of the XF to make adaptations to an other target. It is also possible to let the XF run with an RTOS. This gives the possibility to run state machines in a OS like concurrent manner with threads provided by the underlying RTOS.

  • UnknownGpioState: Just to define the 0 value...
  • GpioRisingEdge: Gpio has gone to high level
  • GpioFallingEdge: Gpio has gone to low state

◆ setState()

void Gpio::setState ( Gpio::eGpioState  state = UnknownGpioState)

Set the state of the gpio.

When an activity is detected on the current gpio, the GpioController call this method to change the current state of the gpio. The gpio's state is ONLY changed if the new state is different from the current one.

Parameters
stateThe new state. (Default value: Gpio::UnknownBtnState)

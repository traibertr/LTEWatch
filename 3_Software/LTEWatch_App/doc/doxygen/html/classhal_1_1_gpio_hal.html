Detailed Description

Class of Hardware Abstraction Layer to handle the different GPIO's of the Nordic nRf5 series chips.

Class of Hardware Abstraction Layer to handle the different GPIO's of the nRf5 series chips based on Nordic SDK v15.0.0 and later. This file contains all functions necessary for the use of any GPIO. It starts with the function gpioInit() to initialize all GPIO to disable unwanted current consumptions.

Initialiaze all GPIOs.

Setup all GPIOs to disable unwated current consumptions. The default value at startup is :

  • All GPIO as Input
  • disconnected
  • pulldown or puulup disabled
  • SENSE disabled

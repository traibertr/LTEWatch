Detailed Description

Reactive class implementing a behaviour.

Reactive class implementing a behaviour. This class can be used to implement a state machine behaviour or an activity behaviour.

Override the processEvent() operation in the inherited class to implement your state machine behaviour.

Push the event in the standard queue.

Push the event in the standard queue. The events contained in this queue will be handled as soon as the critical queue is empty. If the critical queue won't be empty, one event of the standard queue will be handled for each priorityLevelerThreshold critical events.
